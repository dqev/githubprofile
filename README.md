## 🚀 Everything It Does (Features Showcase)

Our generator provides a highly polished editor workspace with a set of modular sections:

### 1. Profile Essentials
- **Personal Details**: Fields for name, role/subtitle, header banner URL, and long biography details.
- **Custom Alignment**: Toggle alignment for the entire intro block (**Left**, **Center**, or **Right**).
- **Avatar Option**: Separate toggle checkbox to show/hide the profile picture avatar in the README output.

### 2. Interactive Tech Stack Grid
- **Interactive State**: Icons are rendered in a clean minimalist grid, showing in grayscale until selected or hovered (which scales and colors them).
- **Badges vs Icons**: Generate stack sections as custom colored Shields.io badges (supporting flat, flat-square, social, plastic, and for-the-badge styles) or a unified SkillIcons grid.
- **Searchable List**: Live text filtering across all languages, frameworks, and utilities.

### 3. Connect with Me (Social Links)
- **High-Resolution Badges**: Instant social indicators for GitHub, LinkedIn, Twitter/X, Discord, Dev.to, Medium, Twitch, and more.
- **Platform Badges**: Support for coding platforms like LeetCode, HackerRank, Codeforces, TopCoder, and Kaggle.
- **Un-squished Icons**: Scaled to keep their natural proportional width ratios.

### 4. Repository & Project Showcase
- **GitHub API Integration**: Input a username and fetch active repositories instantly.
- **Checklist Drawer**: Check off which repositories to include, and the editor builds a clean Markdown table automatically.
- **Custom Projects**: Add custom inputs for non-GitHub or private works.

### 5. Widgets & Profile Stats
- **Stats Graphs**: Display languages breakdown and general repository stats graphs.
- **Streak Tracker**: Show commit streaks.
- **Trophies**: Display GitHub Profile Trophies.
- **View Counters**: Add visitor views badge indicator.
- **Widgets Theme**: Choose from Tokyonight, Dracula, Gruvbox, Onedark, Synthwave, Radical, and High Contrast.

### 6. Alignment & Styling
- **No Borders Layout**: Clean vertical column separator ("cut-from-corners" style) and header alignment.
- **Dual Pane Viewport**: Live Markdown renderer replicating GitHub's default layout side-by-side with raw Markdown code.

## 💻 Technical Implementation
- **Core Stack**: HTML5, Vanilla JavaScript, CSS3 variables.
- **Compiler**: Marked.js (for HTML rendering).
- **Visuals**: FontAwesome & Google Fonts (Inter, Roboto Mono).

## ⚡ Quick Start
No building process or servers required. Clone the directory and open `index.html` in your browser.
```bash
git clone https://github.com/dqev/githubprofile.git
open index.html
```
